# **App Name**: Lumina Finance

## Core Features:

- Intelligent Spending Dashboard: A clean visual interface providing key insights into monthly spend patterns using interactive charts.
- Categorized Expense Logging: Easily input and organize daily transactions into user-defined categories.
- AI Spending Insight Tool: Generative AI tool that analyzes spending behavior to recommend personalized budget optimizations and detect anomalies.
- Budget Goal Setting: Define and track monthly savings goals with progress indicators for each category.
- Automatic Smart Tagging: Utilizes AI to automatically categorize transactions based on merchant description and historical data.
- Data History Feed: A searchable, filterable log of all past financial activities for detailed review.

## Style Guidelines:

- Primary Color: Soft Violet (#BA9CFF), selected for its modern and prosperous feel in a dark mode context.
- Background Color: Deep Obsidian (#131119), a heavily desaturated version of the primary hue to create depth and focus.
- Accent Color: Electric Azure (#819DFF), an analogous hue shifted 30 degrees to the left for striking contrast on financial data points.
- Font pairing: 'Space Grotesk' (sans-serif) for tech-forward headlines and numeric data, and 'Inter' (sans-serif) for readable body text.
- Minimalist, thin-line monochrome icons to maintain a sophisticated and clean aesthetic.
- A grid-based dashboard with cards and soft rounded corners to group related insights clearly.
- Smooth, subtle progress-bar transitions and micro-interactions on hover for a high-end feel.
